\textbf{Internal variables:}\hfill\\
$\coldlist = \{(P^\cold, \ek, \dk, \texttt{leaked}, \texttt{tampered}, \texttt{corrupted}, \texttt{allowc})\}$, a table with the state of each cold storage's key material.\\
$\clientlist = \{(C, \mathcal{I}, t, y, \com)\}$, a table of registered clients and their metadata.\\
$\hotlist = \{(P^\hot, C_j, \hx, \pi, \texttt{time}, \texttt{leaked}, \texttt{tampered})\}$, which keeps track of the key material of every hot storage for each of its clients, and whether it has been leaked or tampered with.\\
$\hotstates = \{P^\hot, \texttt{corrupted}, \texttt{allowc}\}$, which keeps track of whether each hot party has been corrupted.
